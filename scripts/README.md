# scripts/

Developer and CI helper scripts. Keep them small and composable.

- **setup-ai.sh** — creates the Python 3.11 virtualenv for `services/ai-concierge`
  and installs its requirements.
- **dev-ai.sh** — runs the AI concierge service from its venv with reload.
